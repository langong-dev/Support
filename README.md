# Install: 

Install "LanFilePackage" first. https://github.com/langong-dev/LanFilePackage/

Then install "node", "npm", "electron"(npm install electron --save-dev).

```bash
./LanGong install github langong-dev/Support
```

# Run:

```bash
./LanGong run Support
```
